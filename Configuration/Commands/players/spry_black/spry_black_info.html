Player - Black (SPRY/HTML/CSS)

The Player - Black Album Player currently does not support these attributes in the Create Slideshow Generator:

AlbumBook:

  • Allow Clicking Images

Album Properties:

  • Thumbnail
  • Background

Slideshow Properties:

  • Transitions
  • Display Sequence

Additional Notes:

  • Max Image Width should not exceed 500 pixels
  • Previewing in browser locally may not work as expected. Upload files to a web server to preview the slideshow.
